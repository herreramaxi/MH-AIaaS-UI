Datasets

Collection of data that is stored within the system for use in machine learning tasks. It serves as a data source, making it easier to manage, access, and process the data within your machine learning workflows.
Workflows

It defines a pipeline that is composed by a sequence of operations to be applied to a dataset to build a machine learning model.
Models

Built by the execution of a workflow, a ML model captures the underlying structure of the data and can be used to make predictions, classifications, or gain insights on new, unseen data.

Endpoints

It allows clients to consume a machine learning model that was previously run and published.
