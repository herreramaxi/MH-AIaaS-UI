Welcome to Artificial Intelligence as a Service (AIaaS)

AIaaS is designed to facilitates the process of machine learning model generation, allowing users to generate ML models graphically and seemingly deploy it to the Cloud.

Datasets

Collection of data that serves as a data source for machine learning tasks and is used to train models.
Workflows

Pipeline definition consisting of a sequence of operations applied to a dataset to construct a machine learning model.
Models

Produced by workflows, it outlines the necessary steps to convert input data into a prediction.

Endpoints

It allows clients to consume a machine learning model that was previously run and published.

Input a dataset, define your workflow, generate a machine learning model and deploy it to the Cloud!!!
